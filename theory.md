**Метод func не может менять объект, для которого вызыван**
```
class Name
{
  double func() const;    
}
```
**Два двоеточия определяют область видимости. func в области видимости класса Name**
```
double Name::func() const
{
  ...
}
```
**Вызов**
```
Name t;
double result = t.func();
```
